# AFK Footwear E-Commerce Website

AFK Footwear is a responsive e-commerce web application designed for footwear sales.  
This project includes a customer-facing storefront and an admin dashboard for managing products, orders, customers, and reports.

## Features

### User Side
- Product listing by category (Men, Women, Kids, Discounts)
- Product detail page
- Shopping cart
- Checkout process
- Order tracking
- Responsive design (mobile & desktop)

### Admin Dashboard
- Admin authentication (login/logout)
- Product management (CRUD)
- Order management
- Customer data management
- Sales reports
- Returns management
- Clean and structured admin interface

## Tech Stack
- HTML
- CSS
- JavaScript
- PHP
- MySQL
